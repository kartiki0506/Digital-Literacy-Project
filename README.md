## 📝 Module Summaries

### Task 1 – Digital Literacy Awareness Infographic
Created a one-page digital literacy infographic using **Canva**,
covering what digital literacy means, useful digital tools for students,
and tips for building a professional online presence.

### Task 2 – Student Digital Portfolio
Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**.
Created a GitHub profile README and updated the LinkedIn Education section.
Screenshots and a reflection note are included in the task folder.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner-level Python challenge on **HackerRank** and built
a 5-question **Digital Literacy Awareness Quiz** using Google Forms,
linked to a Google Sheet for response tracking.

### Task 4 – Professional Email & Etiquette Guide
Drafted two professional emails — one requesting an assignment extension
and one expressing internship interest. Also created a Social Media
Do's and Don'ts checklist for responsible digital communication.

### Task 5 – Cybercrime Awareness: Ransomware
Researched ransomware using the 2021 AIIMS Delhi cyberattack as a
reference. Produced a detailed case study and a 10-point Stay Safe
Online prevention checklist tailored for college students in India.
